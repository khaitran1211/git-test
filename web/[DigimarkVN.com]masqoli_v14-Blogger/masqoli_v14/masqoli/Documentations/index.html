Getting started

I assume you have a blogger account, so let's get started!

Installation

You can do it by follow this step:

  1. Unzip the “Masqoli_1_4.zip” file.
  2. On Blogger Dashbord Click Template.
  3. Click Backup/Restore button (Top Right).
  4. Click Choose File button. Find where the “Masqoli_1_4.xml” file location.
  5. Then Click Upload.
  6. Important: For Best work copy code from Masqoli_1_4.xml file and paste direct it into the Edit Html Box.


Default Setting

After installing the theme, this it the minimun setting to make all widget works fine in this themes.

Embedded Comment

This to make comment area like the Demo

  2. Click Posts and Comments
  3. in Comment Location Choose Embedded
  4. Then Click Save settings.

Your Youtube video channel gallery



To add this plugin follow this step

  1. On Blogger Dashbord Click Template
  2. Click Edit HTML
  3. Click Format Template
  4. Click Ctrl+F from keyboard
  5. Now search following link:

    6. http://masqoli.blogspot.com/search/label/Video
    Remember: Only change masqoli.blogspot.com with your url. You will find it 3 or 4 times
  6. After replacing again search following link:

    7. http://www.youtube.com/user/Google
  7. Full Javascript code

    	<script type='text/javascript'>
	
      var youTubeChannelURL = "http://www.youtube.com/user/Google";
	
	//optional parameters-----------------------------------------------
	var youTubePlaylistURL = "";
	var youmaxDefaultTab = "uploads";
	var youmaxColumns = 3;	
	var youmaxWidgetWidth = 900;
	//var youmaxWidgetHeight = ;
	var showFeaturedVideoOnLoad = false;
	var showVideoInLightbox = false;

	
	function goClicked() {
		$('#youmax').empty().append(' loading ...');
		youTubeChannelURL=$('#youTubeChannelUrl').val();
		youTubePlaylistURL=$('#youTubePlaylistUrl').val();
		youmaxFeaturedPlaylistId = null;
		prepareYoumax();
	}

	</script>
Property Description
http://www.youtube.com/user/Google change http://www.youtube.com/user/Google with your youtube channel url.
youTubePlaylistURL = A playlist from youtube channel to display its videos in the featured tab. If this is not supplied the latest playlist in your youtube channel is displayed in the featured tab as the featured playlist.
youmaxDefaultTab = A default tab to be selected on load of website. Accepted values can be "uploads" or "playlists" or "featured". If this is not supplied, the featured tab will be selected on load of website.
youmaxColumns = The number of columns (for video thumbnails) to be displayed in the body of the widget. If this is not supplied, a 2-column layout will be selected and displayed.
youmaxWidgetWidth = The width of the Youmax widget. If this is not supplied, width of Youmax defaults to 640px.
youmaxWidgetHeight = The height of the Youmax widget. Ideally this value should not be supplied. If this is not supplied, the Youmax widget takes the minimum height it needs to display all video thumbnails from you youtube cahnnel.
showFeaturedVideoOnLoad = Can be set to true/false. If set to true, the first video in your featured playlist will be auto loaded in the video player embedded in the Youmax widget. If this is not supplied, the value defaults to false.
showVideoInLightbox = Can be set to true/false. If set to true, videos will open in a lightbox. If set to false, videos will open embedded in the Youmax widget. If this is not supplied, the value defaults to false.

Related Posts

  2. Click Edit HTML

  4. Click Ctrl+F from keyboard
  5. Now search following code:

    6. var maxresults=8;

(Default) Complete Javascript to call Related Post like this code: 

        var defaultnoimage="";
        var maxresults=8;
        var splittercolor="#d4eaf2";
        var relatedpoststitle="";
Property Description
var maxresults=8 change the 8 value from maxresults=8 with the number of posts you want to be displayed.
var defaultnoimage=" "; Here between the " " comma you can set a Backup image if post don't have image
